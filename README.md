# Crucible #

A simple Rails app for evaluating FHIR.

## Details ##
  - Ruby 2.0.0+
  - Rails 4.1+
  - Devise
  - Bootstrap
  - Ember.js
  - MongoDB

## Getting Started ##

### OSX ###
- Checkout Crucible:
- Install dependencies with Bundler: ```bundle install```
- Start MongoDB: ```mongod```
- Launch the Rails server: ```bundle exec rails server```
